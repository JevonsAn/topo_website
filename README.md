## Topo信息查询后端
项目使用python3编写，当前部署在10.10.11.141:/home/ayh/new_topo_website下  
使用pipenv进行python包管理：
  - 使用 “pipenv install 包名” 进行包的安装，安装的包会在虚拟环境中
  - 使用 “pipenv shell” 进入当前项目的虚拟环境
  - 以上命令须在项目文件夹下输入
  - 更多命令参考 “pipenv --help”

start_server.py为整个程序的入口，运行项目的命令为：python3 start_server.py  
所有设置均在setting文件夹下
